# Takenoko

## Description 

Takenoko est un tactical RPG (style auto chess) où la gestion des actions (type déplacement et combat) est appliqué de manière autonome.

## Installation

Sous Unix : 
```
sudo apt-get install python3 
sudo apt install python3-pip 
pip3 pygame
```

Sous Windows : 
- Installer Python3 via l'executable du dossier Windows si il n'est pas déjà installé
- Puis dans un invite de commande Windows : 
```
py -m pip install pygame
```
    
## Execution 

Sous Windows :  
```
py chemin_absolue_du_projet\main.py
```
ou
```
cd chemin_absolue_du_projet\
py main.py
```

Sous Unix : 
```
python3 chemin_absolue_du_projet/main.py
```
ou
```
cd chemin_absolue_du_projet/
python3 main.py
```
Pour effectuer le tour d'une unité, il vous suffit d'appuyer sur la barre espace.
Pour bouger la camera on peut appuyer sur les flèches directionnels ou mettre le pointeur de la souris en bordure de la fenetre dans la direction voulue. 
On peut zoom/dezoom avec la molette de la souris .
Les actions effectué par l'unité seront affiché sur le terminal.

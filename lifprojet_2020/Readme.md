# Takenoko

## Description 

Takenoko est un tactical RPG (style auto chess) où la gestion des actions (type déplacement et combat) est appliqué de manière autonome.

## Installation

Sous unix :
dans un shell 
- faire : sudo apt-get install python3 
- faire : sudo apt install python3-pip 
- faire : pip3 pygame

Sous windows : 
- executer l'executable du dossier Windows ( installer python3 si pas déjà installer)
- dans un cmd de windows faire la commande : 
    py -m pip install pygame
    
## Execution 

Sous windows : 
Dans un cmd : 
- faire : py chemin_absolue_du_projet\main.py
ou
- faire : cd chemin_absolue_du_projet\
- puis faire : py main.py

Sous Unix : 
Dans un shell :
-faire : python3 chemin_absolue_du_projet/main.py
ou
- faire : cd chemin_absolue_du_projet/
- puis faire : python3 main.py
